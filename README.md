# 📚 Ders Planlayıcı — PWA

Telefonda uygulama gibi çalışan ders programı uygulaması.

---

## 🚀 Vercel'e Yükleme (Adım Adım)

### 1. GitHub Hesabı Aç
- [github.com](https://github.com) → Sign up → Ücretsiz hesap oluştur

### 2. Yeni Repo Oluştur
- Giriş yaptıktan sonra sağ üstteki **+** → **New repository**
- Repository name: `ders-planlayici`
- **Public** seç
- **Create repository** tıkla

### 3. Dosyaları Yükle
- Repo sayfasında **"uploading an existing file"** linkine tıkla
- Bu ZIP dosyasını çıkart, içindeki TÜM dosya ve klasörleri sürükle-bırak
- **Commit changes** tıkla

### 4. Vercel'e Bağla
- [vercel.com](https://vercel.com) → **Sign Up** → **Continue with GitHub**
- **Add New Project** → GitHub reposunu seç (`ders-planlayici`)
- Hiçbir ayarı değiştirme, direkt **Deploy** tıkla
- 1-2 dakika bekle ✅

### 5. Telefona Ekle
Vercel sana bir link verir: `ders-planlayici.vercel.app`

**iPhone:**
Safari'de linki aç → Alt menüdeki **Paylaş** butonu → **"Ana Ekrana Ekle"**

**Android:**
Chrome'da linki aç → Sağ üst **⋮** → **"Ana ekrana ekle"**

Artık uygulama gibi tam ekran açılır! 🎉

- 📅 Haftalık ders programı
